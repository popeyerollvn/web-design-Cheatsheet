Css.5 Layout and Positioning

css layout and position

TAG Cú Pháp Công Dụng Ví Dụ
border-radius border-radius: 5px; bo vien border border-radius: 5px;
overflow hidden,scroll,visable(default) The overflow property controls what happens to content that spills, or overflows, outside its box
visibility visibility: hidden; visible; tang hinh An element with display: none will be completely removed from the web page. An element with visibility: hidden, however, will not be visible on the web page, but the space reserved for it will.
position relative position: relative; position an element relative to its default static position on the web page. position: relative;
top: 20px;
left: 50px;
will be moved down 20 pixels and to the right 50 pixels from its default static position
Position: Absolute Position: Absolute; When an element’s position is set to absolute all other elements on the page will ignore the element and act like it is not present on the page
Position: Fixed position: fixed; the element will scroll with the rest of the document when a user scrolls. position: fixed;
display display: block;
display: inline;
display: inline-block
z-index z-index: (-n) or n integer; The z-index property controls how far “back” or how far “forward” an element should appear on the web page when elements overlap z-index: 1;
float float: left or right; left - this value will move, or float, elements as far left as possible. right - this value will move elements as far right as possible. float: right;
This works for static and relative positioned elements
